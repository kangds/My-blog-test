My-blog-test
============

完善页面，用于班级内部交流的小型博客
欢迎指导，我想把这个当成我的毕业设计哟

变量定义说明：
user:
 email:  注册邮箱，登录ID（注册后不可更改）
 name:姓名
posts:
 location:出生地（注册后不可更改）
 city:目前所在城市
 hobby:爱好
 phone:电话
 age:年龄
 sex:性别（注册后不可更改）
 sno:学号（注册后不可更改）

 article:
 title:(题目)
 time：写作时间
 authoe:博主
 content:（正文）
 commit:评论（对应应该是嵌套表）

commit:
ci
cauthor:评论作者
ccommit:评论内容
ctime:评论时间

session对应变量名
uid 用户id
cid 评论id
aid 文章id
